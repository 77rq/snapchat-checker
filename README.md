
# Snapchat Checker V1
اداة فحص يوزرات سناب شات فحص دقيق خالي من اليوزرات المبندة !
بالتعاون مع المبرمج Fostn
# الميزات الحصرية
- فحص المبندات وحذفها
- ارسال الصيد على تليقرام
- لا تحتاج الى بروكسي
# التثبيت
- يمكنك تحميل الاداة على جهازك :
```
سيتم تثبيت المكاتب تلقائيا من خلال الاداة ستحتاج الى بايثون فقط
```
